Forms

You can wrap input inside label, quicker than doing 'for'

Look up form action!

Form child elements are all inline!

Forms have a lot of defaults, and browsers style them differently.There is a lot of overriding to be done.

You can use attribute selector [type="text"]. You can also e.g. select things that have placeholders set in this way.

  • Avoid placeholder values as labels
  • Avoid unnecessarily long forms. Use multi-step forms.
  • Do inline form validation
  • Custom form elements must be accessible.
